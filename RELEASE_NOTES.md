# X3-Lights 1.2.1

X3-Lights is back for Satisfactory 1.2. This clean rebuild uses current vanilla light assets while preserving the original mod reference and legacy asset paths for save compatibility.

Version 1.2.1 fixes a crash when a flood light was connected to power by inheriting the current vanilla pooled-light implementation directly.

Requires SML 3.12.0 or newer and game build 491125 or newer.
