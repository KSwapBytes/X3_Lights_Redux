# X3-Lights Redux 1.0.0

X3-Lights Redux brings the original cheaper, lower-power vanilla lights to Satisfactory 1.2 under the unique `X3LightsRedux` mod reference.

The package metadata declares numeric version `1`, matching semantic version `1.0.0` as required by SMR.

This release includes five buildings, four power links per building, and the original Tier 2 unlock. It also fixes the powered flood-light crash by inheriting the current vanilla pooled-light implementation.

## Migrating from X3-Lights

The package redirects legacy `/x3_lights/` assets to `/X3LightsRedux/`. Remove the original package before installing Redux and back up your save before opening it for the first time.

Requires SML 3.12.0 or newer and game build 491125 or newer.
