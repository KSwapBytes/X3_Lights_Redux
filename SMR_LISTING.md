# X3-Lights Redux

X3-Lights Redux updates the original X3-Lights / Cheap Lights mod for Satisfactory 1.2 and SML 3.12 while keeping its familiar vanilla-inspired style.

It includes five Tier 2 buildables:

- X3 Street Light
- X3 Ceiling Light
- X3 Flood Light Tower
- X3 Wall-Mounted Flood Light
- X3 Light Control Panel

The lights use less power than their vanilla counterparts, have cheaper build costs, and provide four power links. This Redux release also fixes the powered flood-light crash by using the current vanilla pooled-light implementation.

## Save Migration

Redux uses the unique mod reference `X3LightsRedux`. It includes redirects for legacy `/x3_lights/` assets so existing buildings and unlock references can migrate from the original mod.

Remove the original X3-Lights package before installing Redux, do not install both versions together, and back up your save before opening it with Redux for the first time.
